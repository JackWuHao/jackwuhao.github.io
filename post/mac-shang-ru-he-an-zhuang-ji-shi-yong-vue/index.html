mac 上如何安装及使用vue

2019-09-16

最近在看vue ,记录一次安装过程避免以后再踩坑

1.需要先安装node node.js 官方下载

2.查看node 版本:

node -v

3.查看npm版本：

npm  -v

4.全局安装依赖包mac下需要加sudo命令:

sudo npm install -g webpack
  1. 全局安装vue-ci 同上：
sudo npm install webpack -g vue-cli

6.查看vue版本

vue  -V

7.创建项目（后面的的项目名字vue-test 可替换成你的）

cd  XXXX(你要创建的目录)
vue init webpack  vue-test

8.然后一路enter键，需要填的填下，不想填能跳过的直接跳过 9.安装依赖

cd   vue-test (你创建的项目名称)
npm install

10.启动项目 (测试环境，本地自带node服务器)

npm run dev

11.发布项目（正式环境）

npm run build
